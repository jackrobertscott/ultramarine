# Padding

Create padding on the element.

### Usage

```jsx
<Element
  padding="slim"
/>
```

```jsx
<Element
  padding="10px 3em 20px"
/>
```

```jsx
<Element
  padding={30}
/>
```

```jsx
<Element
  padding={{
    left: 'mega',
    bottom: 30,
    top: '20px',
    right: '20rem'
  }}
/>
```
