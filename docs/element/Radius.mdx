# Padding

Create corner radius on the element.

### Usage

```jsx
<Element
  radius="slim"
/>
```

```jsx
<Element
  radius="10px"
/>
```

```jsx
<Element
  radius={30}
/>
```

```jsx
<Element
  radius={{
    left: 'mega',
    right: '20rem'
  }}
/>
```
