# Margin

Create margin on the element.

### Usage

```jsx
<Element
  margin="slim"
/>
```

```jsx
<Element
  margin="10px 3em 20px"
/>
```

```jsx
<Element
  margin={30}
/>
```

```jsx
<Element
  margin={{
    left: 'mega',
    bottom: 30,
    top: '20px',
    right: '20rem',
  }}
/>
```
